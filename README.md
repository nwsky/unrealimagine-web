# UnrealImagine Portal

A Next.js MVP website package for **UnrealImagine.com**, positioned as a premium U.S.-based micro-drama production company portal.

## Included in this MVP

- Homepage
- About page
- Works listing page
- Project detail page template
- Capabilities page
- Contact page with inquiry form UI
- Responsive premium dark visual direction

## Run locally

```bash
npm install
npm run dev
```

Open:

```bash
http://localhost:3000
```

## Build

```bash
npm run build
npm run start
```

## Structure

- `src/app/page.tsx` — homepage
- `src/app/about/page.tsx` — about page
- `src/app/works/page.tsx` — works list
- `src/app/works/[slug]/page.tsx` — project detail template
- `src/app/capabilities/page.tsx` — capabilities page
- `src/app/contact/page.tsx` — contact page
- `src/app/data.ts` — editable content data
- `src/app/globals.css` — full styling
